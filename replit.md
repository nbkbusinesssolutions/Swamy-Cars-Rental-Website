# Swamy Cars Rental

## Overview
A car rental website for Swamy Car Rental in Goa, India. This is a static frontend site built with HTML, CSS, and JavaScript, using Vite as the build tool.

## Project Structure
- `index.html`, `about.html`, `cars.html`, `booking.html`, `faq.html`, `contact.html`, `404.html` - Main HTML pages
- `assets/` - JavaScript and CSS assets
- `images/` - Image assets
- `data/` - JSON data files (fleet info)
- `components/` - Reusable components
- `favicon/` - Favicon files

## Development
- **Dev server**: `npm run dev` (runs on port 5000)
- **Build**: `npm run build` (outputs to `dist/`)
- **Preview**: `npm run preview`

## Technology Stack
- Vite 7.3.0 for development and build
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Deployment
Configured as static site deployment with build output in `dist/` directory.
